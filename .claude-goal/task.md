# 任务：优化对话界面的富文本渲染与交互能力

## 任务描述

当前 Claude Code Console 的对话界面仅使用 `escapeHtml()` + `white-space: pre-wrap` 展示原始文本，缺乏任何富文本渲染能力。需要对标主流 Agent 交互界面（ChatGPT、Claude.ai、Cursor、Windsurf、Cline 等），补齐以下核心能力：

1. **Markdown 渲染** — 标题、加粗、斜体、列表、链接、表格等基础 Markdown 元素
2. **代码块高亮** — 围栏代码块带语言标识、语法着色、行号（可选）、一键复制
3. **内联代码** — 单独样式区分
4. **HTML 内容识别与安全渲染** — 检测到 HTML 片段时做沙箱渲染或结构化展示
5. **图片上传与展示** — 支持在输入区域上传/粘贴图片，消息中展示图片（base64 或 URL）
6. **思考过程折叠** — thinking 内容默认折叠，可展开
7. **工具调用结构化** — 更好的卡片化展示
8. **数学公式（可选）** — LaTeX 渲染（如有 Claude 输出含公式）

## 最终目标

- [ ] Claude 回复中的 Markdown 完整渲染（标题/加粗/列表/链接/表格/引用块）
- [ ] 围栏代码块带语法高亮 + 语言标签 + 复制按钮
- [ ] 内联代码有 monospace 背景样式
- [ ] 用户可在输入框粘贴/拖拽/点击按钮上传图片，图片以 base64 随 prompt 发送
- [ ] 消息中的图片（base64 或 URL）正确渲染为 `<img>` 可点击放大
- [ ] 思考过程（thinking）默认折叠为一行摘要，点击展开全文
- [ ] 整体视觉保持当前深色主题风格一致性
- [ ] 无需引入构建工具（保持 CDN + 单文件前端架构）
